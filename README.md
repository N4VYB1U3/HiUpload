# HiUpload

HiUpload is a file upload SaaS that was made using Vue.js [Front-end] &amp; Laravel [Back-end]

## Installation

```bash
npm i
```

```bash
composer install
```

```bash
php artisan key:generate
```

```bash
php artisan migrate
```

## Usage

### Front-end
```bash
npm run serve
```

### Back-end
```bash
php artisan serve
```
